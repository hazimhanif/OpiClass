How To Use

1. Choose the application that you want to check its opinions from Google Play store website.

2. Copy the URL of the Android application download page.

Example URL: https://play.google.com/store/apps/details?id=com.android.chrome

3. Open OpiClass and navigate to the Spam Detection section.

4. Paste the URL of the download page copied earlier.

5. Hit the Get Em! button.

6. Wait for the system to process the request. May take up to 10-15 minutes.

7. View the results when it is ready.

Opinion Spam Detection


About

The phenomenon of opinion spamming raises concerns in the opinion mining domain whose job it is to detect and filter any review spams. Some of the early approaches used in detecting opinion spams were based on manual judgements but this has become outdated because manual judgements are unreliable. The situation has become more dangerous and unpredictable as more and more opinion spams seem to be emerging, making comments about the respective mobile applications. Thus, an automated approach is needed to detection spam opinions.

OpiClass is a prototype development of a web-based opinion spam detection system. It is developed to fulfill the requirement of the candidate master's degree. This prototype adopts a supervised boosting model that uses statistical-based features to detect opinion spams. Currently, the prototype only features the detection of opinion spams for Malay language opinions from Google Play store.
